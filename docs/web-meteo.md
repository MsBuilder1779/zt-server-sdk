# Web Meteo в ZT Server SDK

Web Meteo через SDK позволяет Android-приложению получать метеоданные через серверную экосистему ZEvS.

## Возможности

- получение метеоданных по координатам
- передача высоты точки
- выбор типа результата
- поддержка запроса на конкретное время, если это предусмотрено текущей версией SDK
- обработка ошибок сети и доступа

## Использование

Приложение передает координаты, высоту, тип результата и Web Meteo token через публичный механизм SDK.

Не нужно вручную собирать серверный URL, добавлять служебные заголовки или передавать внутренние параметры SDK.
